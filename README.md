# AI Skills Repository

This repository is a central place for reusable AI assistant configuration used across multiple development stacks.

Its main purpose is to maintain and distribute:
- Stack-specific `.cursor` rules and skills (currently Flutter, Unity, and Template)
- Copy scripts that install those rules/skills into target projects
- Companion documentation for setup and maintenance

In short: this repo is the source of truth for your AI coding rules and skills packs (including the shared Template base), so new projects can be bootstrapped quickly with consistent agent behavior.
